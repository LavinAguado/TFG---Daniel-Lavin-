# 09. Conclusiones

## Resultados

- Se centralizó la información clínica y deportiva.
- Se implementaron pacientes, citas, ejercicios, entrenamientos y documentos.
- Se automatizó la generación y entrega de pautas en PDF.
- Se incorporó seguimiento estructurado post-sesión.
- Se aplicaron autenticación, autorización y seguridad a nivel de fila.
- Se creó un resumen clínico reproducible basado en reglas y métricas.

## Cumplimiento del objetivo

El prototipo demuestra que una arquitectura web modular puede integrar los procesos principales de una clínica multidisciplinar y reducir tareas manuales. El alcance académico se ha cumplido mediante una solución funcional y trazable desde requisitos hasta código.

## Limitaciones

- No se ha validado todavía con un despliegue prolongado en producción.
- No se han realizado pruebas masivas con usuarios finales.
- El resumen no utiliza inteligencia artificial externa.
- Algunas vistas y diagramas simplifican el modelo implementado.

## Trabajo futuro

- Despliegue y evaluación en una clínica real.
- Portal o aplicación móvil para pacientes.
- Estadísticas de evolución y adherencia.
- Integración con dispositivos y wearables.
- Mayor cobertura automatizada y pruebas de seguridad.
- Evolución opcional del resumen con modelos de IA bajo controles de privacidad.

## Cierre

TheraTrack no se limita a digitalizar formularios: conecta el trabajo clínico y deportivo en un flujo único, medible y preparado para evolucionar.
