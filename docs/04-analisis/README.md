# 04. Análisis

El análisis parte de la observación de una clínica que utilizaba herramientas separadas para gestión clínica, agenda y planificación deportiva.

## Necesidades detectadas

- Centralizar la información del paciente.
- Evitar duplicidad de datos y tareas manuales.
- Coordinar fisioterapia y preparación física.
- Automatizar la generación y entrega de entrenamientos.
- Recoger seguimiento estructurado después de cada sesión.
- Proteger información clínica sensible.
