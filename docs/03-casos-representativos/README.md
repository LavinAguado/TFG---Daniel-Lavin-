## Valor técnico y funcional

- Parte de una necesidad clínica real.
- Combina datos de pacientes, profesionales y ejercicios.
- Incluye transacción de base de datos.
- Genera un documento PDF automáticamente.
- Integra un servicio externo de correo.
- Ofrece una ruta pública controlada para el paciente.
- Produce métricas reutilizadas en el resumen clínico.
